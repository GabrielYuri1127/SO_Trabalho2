# Prompt usado no Codex

Criar e organizar completamente um projeto Java para o Trabalho Pratico 2 de Sistemas Operacionais, com duas questoes:

1. Produto escalar sequencial e paralelo usando `Thread`, experimentos com tamanhos e quantidades de threads, medicao com `System.nanoTime()`, aquecimento da JVM, CSV e speedup.
2. Simulacao de uma clinica veterinaria em tempo real, com versao sequencial e versao com threads usando `PriorityBlockingQueue`, `AtomicInteger`, prioridade para emergencias e `Thread.sleep`.

O projeto deve usar Java sem `package`, conter scripts de compilacao e execucao, README, relatorio e referencias, mantendo os integrantes informados.
